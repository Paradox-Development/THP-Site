# Welcome, Debra

This is your website. To make changes, just talk to Claude.

## How to make a change

1. Open this folder in Claude Code.
2. Tell Claude what you want to change. Be as specific or vague as you want — Claude will figure it out and ask if it needs to.
3. Claude will make the change and put it live within a minute.
4. Refresh your browser to see it.

## Want to see a change before it goes live?

Just tell Claude: "Show me what this looks like before publishing."

Claude will give you a link you can click to preview the change locally. Once you approve, tell Claude to publish it.

## If something goes wrong

Tell Aaron. Don't try to fix it yourself. Aaron has a backup of every change ever made and can roll anything back in under a minute.

## What you can't break

Nothing permanent. Every change you make is recorded. If you accidentally delete the homepage, Aaron can restore it. So experiment freely.
